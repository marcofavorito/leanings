# Leanings

Experiments using the Lean theorem prover

## Install Lean
Please follow [this page](https://leanprover-community.github.io/get_started.html)

## Useful commands

```
leanproject new
leanproject add/install
leanpkg upgrade
leanproject build
lean --make path-to-src
```
